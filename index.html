Vapi - A News API

Vapi provide get , post , put and delete methods
That can be easy to use , work fast on your project with Javascript.

Services

The Vapi News API provides a simple , free REST API for developers to retrieve the news in a familiar JSON format

  • No account is required
  • This api servies is absolutely free
  • Retrieve news headlines with different links

Methods

Retrieve the news as valid JSON:

Retrieve all available records using /news endpoint with CORS Headers.

Example

let p = fetch("https://vapi-uxd9.onrender.com/news")
p.then((response) => {
    console.log("response : ", response);
    console.log("response.status : ", response.status);
    console.log("response.ok : ", response.ok);
    console.log("response.header : ", response.header);
    return response.json()
}).then((value) => {
    console.log(value);
}).catch((err) => {
    console.log(err);
})

Retrieve record of given id using /news/:id endpoint with CORS Headers.

Example

let p = fetch("https://vapi-uxd9.onrender.com/news/31")
p.then((response) => {
    console.log("response : ", response);
    console.log("response.status : ", response.status);
    console.log("response.ok : ", response.ok);
    console.log("response.header : ", response.header);
    return response.json()
}).then((value) => {
    console.log(value);
}).catch((err) => {
    console.log(err);
})

Create a new record using /news endpoint with CORS Headers in post method.

Example

let p=fetch('https://vapi-uxd9.onrender.com/news',{
    method:'POST',
    body:JSON.stringify({
        title:'hello',
        news_link:'visg',
        image_link:'http://unplash.com',
        news_host:'vpress'
    }),
    headers:{
        'Content-type':'application/json; charset=UTF-8',
    },
})
p.then((response)=>{
    console.log(response);
    console.log(response.ok);
    console.log(response.status);
    return response.text()
}).then((value)=>{
    console.log(value);
}).catch((err)=>{
    console.log(err);
})

Update a record using /news endpoint with CORS Headers in put method.

Example

let p=fetch('https://vapi-uxd9.onrender.com/news',{
    method:'PUT',
    body:JSON.stringify({
        id:10,
        title:'new news title',
        news_link:'https://facebook.com',
        image_link:'http://unplash.com',
        news_host:'cdn',
    }),
    headers:{
        'Content-type':'application/json'
    }
})
p.then((response)=>{
    console.log(`Response : ${response}`);
    console.log(`Response status : ${response.status}`);
    console.log(`Response ok : ${response.ok}`);
    return response.json()
}).then((value)=>{
    console.log(value);
})

Delete a record with help of news id using /news endpoint with CORS Headers in delete method.

Example

let p=fetch('https://vapi-uxd9.onrender.com/news',{
    method:'DELETE',
    body:JSON.stringify({
        id:8
    }),
    headers:{
        'Content-type':'application/json'
    }
})
p.then((response)=>{
    console.log(`Response : ${response}`);
    console.log(`Response status : ${response.status}`);
    console.log(`Response ok : ${response.ok}`);
    return response.text()
}).then((value)=>{
    console.log(value);
})

*Note : Put and Delete method not work below news id 6.

Methods Response

Code Description
200 Ok
400 Bad Request
500 Internal Server Error

JSON Keys

Each article in response will contain the following keys:

Keys Description
news_id news id
title article title
news_link article link
img_link article image link
news_host name of news source
posted_in publish date and time
